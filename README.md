# Pixijs world map example

This is a demo of an isometric cartoon-like view mesh rendering of any map. It is built on top of [PixiJS](https://pixijs.com/) and uses OpenStreetMap data to create the mesh.


The main logic lies in the `lib/isotoon` directory.


To run the frontend:

```bash
pnpm install
pnpm run dev
```
